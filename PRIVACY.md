# Privacy Policy

CleanTabs is designed to manage tabs locally in the browser.

## Data handled

CleanTabs reads tab URLs, tab titles, tab active state, last accessed time, and tab window/index metadata through Chrome extension APIs.

If the user runs page-text search, CleanTabs temporarily reads visible text content from open web pages to find matches.

## Data storage

CleanTabs does not store tab metadata.

## Data sharing

CleanTabs does not transmit tab data, page content, search queries, or browsing history to any server. CleanTabs does not sell or share user data.

## Permissions

CleanTabs uses the `tabs` permission to list and manage tabs, and the `scripting` permission plus optional site access to search page text when requested by the user.

## Contact

For privacy questions, use the support contact listed on the Chrome Web Store listing or GitHub repository.
